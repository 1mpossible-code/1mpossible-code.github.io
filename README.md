# yemel.me

Static GitHub Pages deployment for [yemel.me](https://yemel.me), Maksym Yemelianenko's personal website.

This repository contains the built site assets served by GitHub Pages from the custom domain configured in `CNAME`.

## Contents

```text
index.html      # GitHub Pages entry point
assets/         # Built JavaScript and CSS bundles
icon.svg        # Site icon
CNAME           # Custom domain configuration
```

## Deployment notes

This repo is the published Pages target, not the main source workspace. Avoid archiving or deleting it unless the domain is moved to another deployment host.
